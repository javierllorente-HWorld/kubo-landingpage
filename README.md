# Kubo Landing Page

Landing page de Kubo, una plataforma de estudio orientada a organizar materias, crear decks y practicar con cards usando repetición espaciada.

El objetivo de este proyecto fue construir una pieza clara y publicada para presentar el producto, explicar su propuesta de valor y mostrar sus principales beneficios de forma simple.

Esta landing acompaña al proyecto principal de Kubo y está pensada como una página comercial de presentación, no como la aplicación en sí.

## Qué comunica la landing

- El problema que busca resolver Kubo.
- La propuesta de valor para estudiantes.
- Los beneficios principales del producto.
- Cómo funciona la plataforma.
- Planes disponibles.
- Preguntas frecuentes.
- Llamado a la acción para empezar o solicitar una demo.

## En qué me enfoqué

- Comunicar el valor del producto de forma clara.
- Diseñar una landing simple, visual y responsive.
- Separar la presentación comercial de la aplicación principal.
- Practicar estructura, copy y navegación de una landing SaaS.
